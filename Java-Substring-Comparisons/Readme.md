## 📖 Java Substring Comparisons

This exercise receives a string `s` and an integer `k`.

The goal is to generate all possible contiguous substrings of length `k`, sort them in lexicographical (dictionary) order, and return:

- the lexicographically smallest substring;
- the lexicographically largest substring.




### Example

**Input**

```text
welcometojava
3
```

**Generated substrings**

```text
wel
elc
lco
com
ome
met
eto
toj
oja
jav
ava
```

**Output**

```text
ava
wel
```
